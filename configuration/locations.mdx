Iftarkar provides location-based time offsets to ensure accurate prayer times across Kashmir's diverse geography. All timings are calculated relative to Srinagar (offset: 0), with adjustments in minutes for other locations.

## How location offsets work

Prayer times are calculated for Srinagar by default. Each district or city has an offset value (in minutes) that is added to or subtracted from Srinagar's timings:

- **Positive offset:** Prayer times are later than Srinagar
- **Negative offset:** Prayer times are earlier than Srinagar
- **Zero offset:** Same as Srinagar

```typescript
// Example: Calculating time for Baramulla (offset: +1)
const offset = Duration.fromObject({ minute: Settings.offset });
const iftarTime = DateTime.fromFormat("18:21", "H:mm").plus(offset);
// Srinagar Iftar: 18:21
// Baramulla Iftar: 18:22
```

  <Step title="Get all available offsets">

    ```typescript
    //   -3: "Pahalgam",

    //   3: "Uri",
    //   4: "Teetwal, Karnah"
    ```

  </Step>
  
  <Step title="Get current location name">
    Use `Settings.offsetLabel` to get the human-readable name for the selected offset.
    
    ```typescript

    ```
  </Step>
  
  <Step title="Find offset by location name">
    Search through the offsets object to find a specific location's offset value.

  </Step>
</Steps>

## Usage in time calculations

The offset is applied when calculating prayer times in `lib/utils.ts:30`:

```typescript
const offset = Duration.fromObject({ minute: Settings.offset });

// Calculate Sehri time
const sehriToday = DateTime.fromFormat(timingsToday.fajr, "H:mm")
  .plus(offset)
  .plus(sehriOffset);

// Calculate Iftar time
const iftarToday = DateTime.fromFormat(timingsToday.maghrib, "H:mm")
  .plus(offset);
```

<Info>
  Offsets are applied using Luxon's `Duration` and `DateTime` classes for accurate time arithmetic.
</Info>

## Geographic rationale

The offset values reflect astronomical differences due to:

- **Longitude:** Locations east of Srinagar see sunrise/sunset earlier (negative offset)
- **Longitude:** Locations west of Srinagar see sunrise/sunset later (positive offset)
- **Altitude:** High-altitude locations like Leh have different horizon angles

<Note>
  These offsets are calculated by Islamic scholars and verified against astronomical data for Kashmir's specific geography.
</Note>
