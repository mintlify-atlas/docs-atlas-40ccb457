Iftarkar provides an intelligent prayer time tracking system with a real-time countdown timer that automatically adapts to the current phase of the day during Ramadan.

## How it works

The timer continuously monitors the current time and calculates the next prayer event (Sehri or Iftar) based on authentic timings for Kashmir.

<Steps>
  <Step title="Automatic phase detection">
    The system detects three distinct phases throughout the day:
    
    - **Pre-Sehri**: Before the fast begins (shows countdown to Fajr)
    - **Fasting**: During the fasting period (shows countdown to Maghrib)
    - **Post-Iftar**: After breaking the fast (shows countdown to next Sehri)
  </Step>

  <Step title="Real-time calculations">
    Prayer times are calculated using the `getTimes()` function which:
    
    ```typescript
    const times = getTimes();
    setTimeEnd(times.next);
    setUpcomingLabel(times.label);
    setPhase(times.phase);
    setTimeLeft(times.next.diffNow(["days", "hours", "minutes", "second"]));
    ```
    
    The timer updates every second to show precise remaining time.
  </Step>

  <Step title="Visual progress tracking">
    A segmented progress bar displays elapsed time between prayer events:
    
    ```typescript
    setProgress(
      (times.previous.diffNow().milliseconds * -100) /
        times.next.diff(times.previous).milliseconds
    );
    ```
    
    This provides visual feedback showing how far into the current phase you are.
  </Step>
</Steps>

## Precision timing logic

The system retrieves timings for three days (yesterday, today, tomorrow) to handle edge cases:

```typescript
const today = now.toFormat("ddLL");
const yesterday = now.minus(oneDayDuration).toFormat("ddLL");
const tomorrow = now.plus(oneDayDuration).toFormat("ddLL");

const timingsToday = Settings.timings[today];
const timingsTomorrow = Settings.timings[tomorrow];
const timingsYesterday = Settings.timings[yesterday];
```

This ensures seamless transitions between days and accurate calculations during overnight periods.

## School-specific adjustments

<Note>
  Different Islamic schools of thought have varying precautionary buffers:

  - **Fiqah Hanafiya**: Standard Fajr time for Sehri end
  
  These adjustments are automatically applied based on your selected method.
</Note>
